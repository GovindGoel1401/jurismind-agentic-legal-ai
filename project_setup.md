Use the following stack:

React
Vite
TypeScript
TailwindCSS
ShadCN UI
React Router
Framer Motion
Axios
Recharts
Lucide Icons

Initialize the project using Vite with React + TypeScript.

Project requirements:

1. Configure TailwindCSS.
2. Install and configure ShadCN UI.
3. Configure React Router.
4. Setup Axios for API communication.
5. Configure environment variables using .env.

Create the following project structure:

src/
  components/
  pages/
  layouts/
  services/
  hooks/
  utils/
  types/
  assets/
  styles/

Important:

Create a proper .gitignore file that excludes:

node_modules
dist
.env
.env.local
coverage
.DS_Store

Create a .env.example file with:

VITE_API_BASE_URL=http://localhost:8000
VITE_APP_NAME=JurisMind AI

Create a README explaining how to:

install dependencies
run the dev server
build production

Do not connect to backend yet. Use mock data.

The UI should follow a legal-tech theme with black, white and gold colors.
